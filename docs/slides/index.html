なぜ検索機能が必要か

PROBLEM

  • 現在はタブ切り替えで表示を絞るしかない
    （自分 / メンバー / 未割当 / 完了）
  • タスク数が増えると目的のタスクを見つけにくい
  • タブをまたいで横断的に検索できる手段がない

SOLUTION

🎯 タスク名・担当者名・キーワードから
素早くタスクを見つけられるようにする
✅ フィルタータブをまたいで全件横断検索

何を検索対象とするか

通常検索
未完了タスク全件（タブをまたいで横断的に検索）
完了済み検索
完了済みタスク全件（「完了済みを検索」タップ時）
削除済みタスク
対象外（ソフトデリート済みは除外）
既存のフィルタータブ・詳細モーダルは変更なし。検索とは独立した機能として共存する。
